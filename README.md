# TASKER

## What for?

**Convert markdown to tasks.**

A task will be created from every list item with a priority set in square
brackets, like below:

```
[1] Task priority 1
[0] Task priority 0
```

Use Live Server in VSCode to see the tasks.

The example is from Bootstrap 5.3 Update
